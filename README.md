# Servidor-Monitoreo
Servidor de monitoreo con NodeJS


Crear una carpeta para mover los arhios de server.js y la carpeta de public
 seguido del comando    ===> npm -y init 
 echo esto instalar :
                               npm install express socket.io systeminformation


Y abrir el tunel con:
                          npx localtunnel --port 3000  <====================> El servidor se ejecuta bajo el puerto 3000
